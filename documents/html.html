La structure du document

<h1> à <h6>
Les titres de la page : <h1> est le titre le plus important
<p>
Un paragraphe
<hr>
Une ligne horizontale
<div>
Une section block anonyme
<span>
Une section en ligne anonyme

La mise en forme

<strong> ou <b>
Mettre en gras
<em> ou <i>
Mettre en italique
<u>
Sousligner
<strike> ou <s>
Barre un texte
<br>
Aller à la ligne
<sub>
Placer du texte en indice
<sup>
Placer du texte en exposant

Les liens

<a href="http://www.example.com">
Un lien vers une page web
<a href="mailto:prenom.nom@univ-lyon3.fr">
Une lien vers l'écriture d'un mail avec le destinataire pré-rempli
<a href="#somewhere">
Un lien vers une partie de la même page web

Les médias

<img src="http://www.example.com/mon-image.png">
Une image
<video><source src="http://www.example.com/ma-video.mp4" type="video/mp4"></video>
Une vidéo

Les listes

<ol>
Une liste ordonnée : Numérote les éléments de la liste
<ul>
Une liste non-ordonnée : Ajoute des puces devant le éléments de la liste
<li>
Un élément d'un liste : Peut être enfant d'un <ol> ou d'un <ul>

La sémantique

<abbr>
Indique une abréviation
<acronym>
Indique un acronyme
<address>
Indique une adresse
<blockquote>
Longue citation
<q>
Courte citation

Les balises pour les formalités

<!DOCTYPE html>
Initialiser un code HTML (pas vraiment une balise, elle n'a pas besoin d'être fermée)
<!-- Comment Text -->
Un commentaire HTML
<html>
Une page HTML
<head>
Les informations sur la page
<title>
Le titre de la page : Ce titre est utilisé dans l'onglet de votre navigateur
<meta>
Une information de la page : Cette information ne sera pas affichée sur le navigateur
<body>
Le contenu de la page
